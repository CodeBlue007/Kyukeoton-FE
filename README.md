# 그로우(Grow) 
- 큐시즘에서 진행한 큐커톤 결과물


## 1. 서비스 개요
### 느린학습자 아동들에게 학습을 도와주는 게임 앱

<hr/>

- 느린학습자란? 지능지수 71~84 사이인 경계선 지능을 가진 아이들을 말함.

- 느린 학습자 아동은 직접적으로 게임에 참여해 문제를 해결 할 수 있음
- 느린 학습자 보호자는 함께 문제 해결을 돕고 보호자 역할을 수행함.

## 2. 기술스택

- FE : React Native, Recoil, axios, styled-components
- BE : Java Spring
